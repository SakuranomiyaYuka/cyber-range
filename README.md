# Cyber Range

一个 Web 安全训练靶场，代码全部原创。适合安全学习者练习常见的 Web 漏洞利用技巧。

> ⚠️ **仅供授权的安全学习和测试使用。请勿用于非法用途。**

## 快速启动

### Docker Compose（推荐）

```bash
docker compose up -d
# 访问 http://localhost:5000
```

### 直接运行

```bash
cd app
pip install flask gunicorn
python app.py
# 访问 http://localhost:5000
```

## 关卡列表

### SQL 注入（5关）

| 关卡 | 类型 | 难度 | 说明 |
|------|------|------|------|
| Lv1 | GET 注入 | ★☆☆ | URL 参数直接拼接，单引号闭合 |
| Lv2 | POST 登录绕过 | ★☆☆ | 登录表单注入，注释符绕过 |
| Lv3 | 布尔盲注 | ★★☆ | 页面只返回存在/不存在状态 |
| Lv4 | 时间盲注 | ★★★ | 通过 SLEEP 延时判断 |
| Lv5 | UPDATE 注入 | ★★★ | 在 UPDATE 语句中注入子查询 |

### XSS 跨站脚本（5关）

| 关卡 | 类型 | 难度 | 说明 |
|------|------|------|------|
| Lv1 | 反射型（无过滤） | ★☆☆ | 直接输出，最简单的弹窗 |
| Lv2 | 反射型（标签绕过） | ★☆☆ | 过滤了 script 标签，可用 img/body 等 |
| Lv3 | 存储型（留言板） | ★★☆ | 评论内容存储在数据库，持久化 XSS |
| Lv4 | DOM 型 | ★★☆ | 前端 innerHTML 写入，不经过服务端 |
| Lv5 | 存储型 + CSP | ★★★ | CSP 限制内联脚本，需找到绕过方式 |

### 文件上传（5关）

| 关卡 | 类型 | 难度 | 说明 |
|------|------|------|------|
| Lv1 | 无过滤 | ★☆☆ | 直接上传任意文件 |
| Lv2 | 后缀黑名单 | ★☆☆ | 黑名单不全，大小写/变体绕过 |
| Lv3 | MIME 检查 | ★★☆ | 检查 Content-Type，客户端可控 |
| Lv4 | 文件头检查 | ★★★ | 检查文件幻数，GIF89a 头部绕过 |
| Lv5 | 多重过滤 | ★★★ | 综合所有检查，需要组合技巧 |

## 技术栈

- **后端**: Python Flask + SQLite
- **部署**: Docker Compose
- **前端**: 原生 HTML/CSS/JS（无第三方依赖）

## 注意事项

1. 首次运行会自动初始化数据库
2. CV 预设了多个测试用户账号（在 db.py 中可查看）
3. 部分 flag 存储在 secrets 表中，需要通过注入获取
4. 上传的文件保存在 uploads/ 目录
